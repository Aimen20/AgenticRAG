# AgenticRAG
This project demonstrates RAG architecture using Gemini, LlamaIndex and Qdrant. It highlights the agentic workflow to decide which action to take next.
# Key Features:

**Agentic Framework:** Gemini to think and take actions

**Multimodal Functionality:** Image and textual inputs to query vector store

**Advanced Vector Search:** A Qdrant Vector Store storing Text and Image Embeddings

**Interactive UI:** A seamless chat experience built with Chainlit
